# Regex Extractor (Python)

A simple Python script that extracts email addresses from a text file using Regex.

## What It Does
- Reads a text file
- Extracts emails using regular expressions
- Removes duplicates
- Saves results to a new file
- Prints count summary

- regex_extractor.py
- sample_input.txt
- extracted_emails.txt
